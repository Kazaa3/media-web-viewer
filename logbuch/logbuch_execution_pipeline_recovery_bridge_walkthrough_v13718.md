# Pipeline Recovery Bridge: Walkthrough (v1.37.18)

## Overview
The Pipeline Recovery Bridge provides a professional-grade "Tactical Kill Switch" for resolving stalled FFmpeg and mkvmerge media streams directly from the forensic overlay.

## Key Features
- **Targeted Media Purge:** Verified, path-aware termination of media processes via `kill_stalled_ffmpeg_streams()`.
- **Forensic UI Integration:** [RECOVER] button and status indicator in the VID (Video Health) pane.
- **SENTINEL Trace Integration:** Persistent forensic logging of all recovery events.

## Usage
1. Open the VID (Video Health) tab in the diagnostics overlay.
2. Click the [RECOVER] button if a stream is stalled.
3. Confirm the action; monitor the status indicator and SENTINEL logs for feedback.
