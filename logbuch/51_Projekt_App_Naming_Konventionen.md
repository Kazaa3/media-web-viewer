# Projekt-App-Naming-Konventionen

Dieses Dokument definiert die offiziellen Header-Namen für die Media Web Viewer Projekt-App und deren Komponenten.

---

## Gültige Header-Namen

- **dict - Desktop Media Player and Library Manager**
  - Für Desktop-Skripte, Desktop-Dokumentation und native Komponenten.
- **dict - Web Media Player and Library Manager**
  - Für browserbasierte Skripte, Web-Dokumentation und Frontend-Komponenten.

Beide Header folgen denselben Konventionen und können in Scripts, Tests, Markdown-Dateien und Codebuch-Einträgen verwendet werden.

---

## Anwendung

- Wähle den passenden Header entsprechend dem Kontext (Desktop oder Web).
- Dokumentiere neue Komponenten, Tests und Skripte mit dem jeweiligen Header.
- Halte dich an die Style Guides und Codebook-Vorgaben.
