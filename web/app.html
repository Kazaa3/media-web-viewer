Media Web Viewer

            
Eine Desktop-Anwendung zur Verwaltung und Wiedergabe von Mediendateien mit
                erweiterten
                Metadaten-Parsing-Funktionen.


            
Technologie

            
Entwickelt mit Python, Eel, Bottle und SQLite.


            
Funktionen

            
Metadaten-Extraktion, Format-Support (MP3, FLAC, M4B), Transkodierung,
                Kapitel-Support.


            
Datenschutz

            
Lokale Speicherung in SQLite. Keine Datenübertragung an externe Server.
            


            
Lizenz

            
Open Source Projekt.


            

            

                Entwickler: kazaa3 | Ort: Deutschland
            

        

        

            
        

    


    
    

        

            
Über Media Web Viewer

            
        

        

            
Media Web Viewer


            
Eine umfassende Desktop-Anwendung zur
                Verwaltung, Indexierung und
                Wiedergabe von Mediendateien mit hochperformanten Metadaten-Parsing-Funktionen.


            
Hauptmerkmale

            
Multi-format support, Intelligent parser chain,
                Audiobook support, Automatic transcoding, SQLite DB, Test suite, Logbook.


            
Zur Zeit verfügbar

            
Library management, File indexing, Audio
                player with chapters, Real-time debug logs.


            
Geplante Features

            
Video playback, Playlist management, Theme
                configuration.


            

            

                Weitere Informationen findest du im Logbuch unter dem "Logbuch"-Tab.
